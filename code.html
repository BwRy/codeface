Community Resources

Source Code Repository

Codeface development takes place on github. Obtain a copy of the repository with
git clone https://github.com/siemens/codeface.git

Although Codeface is already fully usable, installation requires more manual effort than we would like. That's why we have not cut formal releases yet.

Mailing List

Subscribe to the Codeface mailing list by sending an eMail with subject 'subscribe' to codeface-request@freelists.org. The mailing list is located at codeface@freelists.org and the list archive can be found at http://www.freelists.org/archive/codeface.

Project Wiki

The project wiki is hosted on github.

Contributing Code

Currently, Codeface development is driven by the Siemens Corporate Competence Centre for Embedded Linux. Maintainance is performed by Wolfgang Mauerer <wolfgang.mauerer AT siemens.com>. Contributions are highly welcome! Apart from that, the following rules apply:

  • Submissions are preferred in the form of pull requests against the latest state of the master branch, after the changes have been discussed on the mailing list.
  • To track the origin of contributions, we use the Signed-off-by system, so the corresponding lines in commits are mandatory.
  • By submitting patches or pull requests, you agree that your changes are made available under the GPLv2.
  • Information on how to structure commits and source code style guidelines are available in the Wiki.
