Screenshots

Some illustrations of the web-based Codeface analysis workbench.

Concise Overviews

Overviews summarise the evaluation results for various categories of interest, leading to general assessments of projects that highlight strengths and weaknesses.

Detail Summaries

Summaries that justify a classification decision further are available for all analysis categories. Notice that it's possible to dynamically compare projects, which enables users to judge the state of unknown projects compared to well-known examples.

Dig into the raw data

Projects can be examined in great detail depth. This way, the decisions made by the categorisation and classification algorithms can be justified based on objective input data.

Adapt to personal preferences

Views in the web front-end can be dynamically adapted to the personal preferences or evaluation needs. It is possible to re-arrange the display and add/remove widgets without the need for static configuration.
