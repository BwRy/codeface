About Codeface

Codeface is a framework and interactive web frontend for the social and technical analysis of software development projects. It captures various data sources (revision control systems, bugtracking systems, mailing lists and complexity metrics) to provide:

  • An interactive web-frontend to analyse projects on various levels of detail from management summaries (communication, collaboration, construction, complexity) down to the raw data. It support direct comparison of projects.
  • A fully automated, parallelised analysis framework that scales to large projects (including the Linux kernel). New projects can be added with minimal configuration effort.
  • A holistic database that coherently integrates all data sources and allows for complex, insightful queries.
  • A research platform for software engineering using advanced statistical techniques and methods of machine learning on a significant amount of projects.
  • Quantitative analysis and objective classification of social aspects of software engineering.

Features

  • Combine the powers of Python for text processing with the capabilities of R for statistics and machine learning, and the speed of MySQL for handling big data.
  • Detect central contributors based on their social influence, not just on static measures like changed lines of number of commits.
  • Multiple novel approaches to infer collaboration structures from revision control systems.
  • Detection of most active discussion topics and communication structure from mailing list using advanced text mining.
  • Distinguish between active and passive contributors to mailing lists, and evaluate the structure of discussions.
  • Various state-of-the-art clustering algorithms allow for detecting closely interacting communities, and how they change over time. Community quality estimation via Monte Carlo methods.
  • Numerous traditional software engineering metrics resolved by time to detect project evolution and trends.
  • Time series analysis (e.g., growth data, complexity behaviour, communication volume): Estimate trends, determine regularity and self-consistency.
  • Highly interactive web frontend based on reactive programming (Shiny): Technical details of interactive parameter passing are handled by the framework, new visualisations and analyses can be added with little effort.
  • Per-session configurable web frontend: Create analysis panels that reflect the interest of individual users and use-cases.
